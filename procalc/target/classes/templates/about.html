About ProCalc

ProCalc is a Spring Boot + Thymeleaf calculator with a polished, responsive UI.

Features

  • Core ops: +, −, ×, ÷, %; parentheses
  • Scientific: sin, cos, pow, and ^ operator
  • Live preview with adjustable delay
  • History with search, export (JSON/CSV), and pinned favorites
  • Memory: MC, MR, M+
  • Keyboard shortcuts and key highlighting
  • Sound feedback (toggle in Settings)
  • Theme toggle with animated transitions; glassmorphism UI

Tech

  • Java 17, Spring Boot
  • Thymeleaf, Bootstrap 5
  • REST endpoints: /api/v1/evaluate, /preview, /history, /memory

Design & UX

  • Ripple interactions, subtle hover/press, page fade-ins
  • Accessible semantics and focus states
  • Responsive layout with offcanvas history on mobile
