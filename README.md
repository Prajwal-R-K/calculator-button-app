## 🌍 Deployment on Render

1. Push your project to GitHub.
2. Go to [Render](https://render.com).
3. Click **New Web Service** → Connect your repo.
4. Choose **Build Command**:

   ```
   ./mvnw clean package
   ```
5. Choose **Start Command**:

   ```
   java -jar target/*.jar
   ```
6. Deploy → Your app will be live! 🎉
